# Project Command Center

Task 1 bootstraps the initial Next.js application shell for the project management
SaaS MVP.

## Getting started

1. Install dependencies with `pnpm install`.
2. Start the app with `pnpm dev`.
3. Run the unit tests with `pnpm test`.

## Included in this bootstrap

- Next.js App Router foundation
- TypeScript configuration
- Tailwind CSS and PostCSS setup
- Vitest and Testing Library unit test harness
- Root layout shell with the initial product title
